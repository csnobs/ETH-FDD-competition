# ETH-FDD-competition

Competition project for **Week 1 (Information Theory)** of the *From Data to Decisions* module, part of the ETH Zurich CAS in Data and Machine Learning program.

## Setup

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

## Tests

```bash
pytest
```

## Project structure

- `main.py` — entry point
- `src/` — core code
- `data/` — raw datasets (do not edit)
- `tests/` — unit tests
